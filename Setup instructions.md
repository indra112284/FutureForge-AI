Setup Instructions – FutureForge AI
1. Clone the Repository

Download the project from GitHub.

git clone https://github.com/indra112284/FutureForge-AI.git

Navigate to the project folder:

cd FutureForge-AI
2. Install Python Dependencies (Backend)

Install all required Python libraries.

pip install -r requirements.txt
3. Run the Backend Server

Start the backend server using Flask.

python app.py

Backend will run at:

http://127.0.0.1:5000
4. Install Frontend Dependencies

Navigate to the frontend project folder.

cd frontend

Install the required packages.

npm install
5. Run the Frontend

Start the React development server.

npm start

Frontend will run at:

http://localhost:3000
6. Open the Application

Open your browser and go to:

http://localhost:3000
7. How to Use the Application

Create an account using email and password.

Login to access the dashboard.

Start the programming skill assessment quiz.

Submit the quiz to view your score and performance analysis.

The system identifies strong and weak areas.

Based on performance, the platform generates a learning roadmap for the programming language.

Users must clear the current quiz before unlocking the next level.

8. Technologies Used

Frontend:
React.js, HTML, CSS, JavaScript

Backend:
Python, Flask

AI / ML:

Database:
SQLite / SQL

Tools:
